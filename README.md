# Teknik-Sampling-dan-Survei-
Penelitian kuantitatif mengenai pengaruh media sosial terhadap konsentrasi belajar mahasiswa ITSB melalui penyebaran kuesioner Google Form kepada 79 responden. Data dianalisis secara statistik untuk mengetahui dampak media sosial terhadap aktivitas akademik dan fokus belajar mahasiswa di lingkungan kampus.
